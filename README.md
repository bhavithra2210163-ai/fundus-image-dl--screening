# fundus-image-dl--screening
Deep learning pipeline for disease screening using retinal fundus images

## Advanced Model (Research Work)

In my academic/research work, I developed a more advanced deep learning model using:

- ResNeXt50 / EfficientNet backbone
- Feature extraction approach
- Stratified train-validation-test split
- Clinical metadata integration (age, gender)
- Evaluation metrics: Accuracy, F1-score, ROC-AUC

Due to dataset and computational constraints, this repository contains a simplified demonstration version.
